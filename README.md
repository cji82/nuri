# 누리호 발사 시뮬레이터 게임 🚀

실제 누리호 4차 발사 타임라인을 모티브로 한 타이밍 액션 게임입니다.  
각 단계(1단 분리 → 페어링 분리 → 2단 분리 → 페이로드 분리)를 정확한 고도·속도 근처에서 스페이스바로 실행해 누리호를 성공적으로 궤도에 올려보세요.

## 주요 특징

- **실제 데이터 기반 고도·속도 프로파일**: 1단 분리 60 km·Mach 4.5 등 실제 수치에 맞춘 시뮬레이션.
- **액션 힌트 & 허용 시간 창**: 단계가 준비되면 중앙 힌트와 카운트다운(5초)이 떠서 타이밍을 놓치기 어렵습니다.
- **배경 장애물**: 구름, 새, 비행기가 끊임없이 등장하며 충돌하면 게임오버.
- **축하/실패 오버레이**: 모든 미션 성공 시 “대한민국 누리호 발사 성공” 메시지, 실패 시 원인 안내.

## 조작 방법

| 키 | 기능 |
| --- | --- |
| `A / ←` | 로켓 좌측 회피 |
| `D / →` | 로켓 우측 회피 |
| `Space` | 활성화된 단계 실행 |
| `발사 시작` 버튼 | 카운트다운 10초 후 시뮬레이션 시작 |
| `재시작` 버튼 | 즉시 초기화 후 다시 시작 |
| 성공/실패 화면의 `대기화면으로` | 대기 화면으로 복귀 |

## 단계 목표

1. **1단 분리**: 고도 60 km, 속도 Mach 4.5 (≈1530 m/s)
2. **페어링 분리**: 고도 120 km
3. **2단 분리**: 고도 200 km
4. **페이로드 분리**: 고도 700 km

각 단계는 시간·고도 허용 범위 안에 들어오면 5초 동안 “스페이스바!” 안내가 유지됩니다. 정확도에 따라 점수와 성공률이 계산됩니다.

## 실행 방법

```bash
npm install        # 정적 서버가 없다면 임의로 준비 (optional)
python -m http.server 8000  # 또는 아무 정적 서버
# 브라우저에서 http://localhost:8000 접속
```

또는 `index.html`을 브라우저로 직접 열어도 됩니다.

## 개발 노트

- 고도·속도는 프로파일 기반 선형 보간으로 계산하고, 시뮬레이션 속도를 1.4배로 조정해 루즈함을 줄였습니다.
- 점수판은 게임오버/성공 시 자동 초기화되며, 모든 단계 완료 시 축하 오버레이가 표시됩니다.
- 장애물은 0.4~1.3초 간격으로 여러 개가 동시에 생성될 수 있어, 회피 입력이 중요합니다.

즐겁게 플레이하시고, 누리호 발사 성공을 축하합니다! 🇰🇷🚀
